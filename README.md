This project is a web application designed to facilitate medical consultation. It includes informational pages about the procedure, patients testimonials, consultation scheduling and many more. The project uses HTML for structure, CSS and BOOTSTRAP for styling and JAVASCRIPT to make it interactive.
### FEATURES:
-	Information page: Detail information about treatments according to the requirements 
-	Contact Form: Contact the medical team
-	Consultation Form: To schedule a consultation 


### FUTURE ENHANCEMENT:
-	Appointment Confirmation: Integrate with a backend to store and confirm appointments.
-	User authentication: Allows User to create accounts and track their consultations.
-	Advanced Interactivity: Enhance the site with more JavaScript features like maps, chat bots, etc.
